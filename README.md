# 愛知県高等学校吹奏楽コンクール 代表校一覧

## 概要
2024年と2025年の愛知県高等学校吹奏楽コンクール代表校を比較表示する Webサイトです。

## 機能
- 📊 統計情報の表示
- 🔍 フィルター機能（連続代表、新規代表、シード校など）
- 🎨 視覚的に分かりやすいデザイン
- 📱 レスポンシブ対応

## 表示内容
- 地区大会代表校（26校）
- 名古屋地区代表校（7校）
- シード校（2校）
- 連続代表校（5校）

## フィルター機能
- 全て表示
- 連続代表
- 新規代表
- シード校
- 2024年のみ

## 技術仕様
- HTML5 + CSS3 + JavaScript
- Google Fonts（Noto Sans JP）
- レスポンシブデザイン
- アニメーション効果

## 表示方法
`index.html` をブラウザで開いてください。

## データソース
愛知県吹奏楽連盟公式データを基に作成
